ATIVIDADE AVALIATIVA 1- Construção do MVP (Produto Mínimo Viável)
Você deve desenvolver a primeira versão do seu portfólio profissional utilizando HTML5, CSS3 e JavaScript. O projeto deve conter:
•	Estrutura Semântica: Cabeçalho com menu, duas seções (Sobre mim e Meus Projetos) e Rodapé.
•	Estilização: Uso de Flexbox ou Grid, e responsividade básica para dispositivos móveis.
•	Interatividade: Implementar via JavaScript um array de objetos contendo seus projetos e renderizá-los dinamicamente no HTML.
•	Versionamento: O projeto deve ser enviado para um repositório público no GitHub.
•	Publicação: O projeto deve ser publicado utilizando o GitHub Pages.

Por fim, nessa atividade, envie um arquivo PDF com 2 links:
•	Link do seu repositório público do GitHub.
•	Link do seu projeto publicado no GitHub Pages.
